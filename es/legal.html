Cookies

Instalamos cookies de Google Analytics, con la finalidad de analizar la navegación de los usuarios por nuestro sitio web. Para utilizar este sitio web no resulta necesaria la instalación de cookies, puedes no aceptarlas o seguir las siguientes instrucciones en tu navegador para bloquearlas y, en su caso, eliminarlas:

  • Internet Explorer: Ir a Herramientas -> Opciones de Internet -> Privacidad -> Configuración. Para más información, puede consultar el soporte de Microsoft o la Ayuda del navegador.
  • Firefox: Ir a Herramientas -> Opciones -> Privacidad -> Historial -> Configuración Personalizada. Para más información, puede consultar el soporte de Mozilla o la Ayuda del navegador.
  • Chrome: Ir a Configuración -> Mostrar opciones avanzadas -> Privacidad -> Configuración de contenido. Para más información, puede consultar el soporte de Google o la Ayuda del navegador.
  • Safari: Ir a Preferencias -> Seguridad. Para más información, puede consultar el soporte de Apple o la Ayuda del navegador.

A continuación describimos las cookies de Google Analytics:

Nombre de cookie Duración Descripción
_ga 2 años Recopila información anónima. Se usa para distinguir a los usuarios.
__utma Dos años a partir de la configuración o actualización Recopila información anónima. Se usa para distinguir usuarios y sesiones.
__utmb 30 minutos a partir de la configuración o actualización Recopila información anónima. Se usa para determinar nuevas sesiones o visitas.
